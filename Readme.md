# Chaos and Agents Simulation

Collection of simulations inspired by chaos theory and emergent behavior.

## Contents

### Chaos systems
- Hénon attractor
- Julia sets
- Rössler attractor
- Feigenbaum bifurcation tree

### Agent systems
- Worm-like autonomous agents with vision, memory and wall-following behavior

## Tech
Originally implemented in C++ using WinAPI pixel rendering.

## Future work
- Migration to Python
- Interactive visualization (matplotlib / pygame)
- Modular agent framework
